PROJECT PLAN: CORRELATE TRACKER

Complete Technical Specification

Version 3.0 — Implementation Ready

1. EXECUTIVE SUMMARY

A web-embeddable micro-app for tracking personal variables (substances, routines, medications, peptides) and correlating them with subjective or objective outcomes using advanced statistical modeling.

1.1 Core Question

"How does Factor X affect Result Y, and when?"

1.2 Key Design Decisions

  • Stack: FastAPI (Python) + React + PostgreSQL
  • Deployment: Docker-based self-hosted MVP with cloud migration path
  • Access: PWA — installable on phone/desktop, single codebase
  • Auth: JWT with username/password (single user MVP)
  • Analytics: On-demand TLCC calculation (button trigger or post-entry)
  • Notifications: PWA push notifications (iOS compatible)

3. AUTHENTICATION & SECURITY

3.1 MVP Auth Flow

  • Single user: Username/password stored in environment variables or hashed in DB
  • JWT tokens: Access token (15 min) + Refresh token (7 days)
  • Storage: HttpOnly cookies (CSRF protection) or localStorage
  • Password hashing: bcrypt with salt

3.2 Future Multi-User Extension

  • Add users table with role field
  • Add user_id foreign key to all data tables
  • Implement registration endpoint
  • Optional: OAuth2 (Google, Apple) via authlib

3.3 Security Measures

  • HTTPS: Enforced via Caddy auto-TLS
  • CORS: Strict origin whitelist
  • Rate limiting: slowapi on auth endpoints
  • Input validation: Pydantic schemas on all endpoints
  • SQL injection: Prevented via SQLAlchemy parameterized queries

7. ANALYTICS ENGINE

7.1 TLCC Implementation

Time-Lagged Cross-Correlation algorithm in Python:

  1. Resample events and reports to hourly time-series
  2. Compute Pearson correlation at each lag (0h, 1h, 2h, ... 72h)
  3. Identify lag with maximum absolute correlation
  4. Calculate p-value and confidence intervals
  5. Store results in correlations table

7.2 Statistical Safeguards

  • Cold Start: Minimum 7 days of data before showing correlations
  • Sample Size: Minimum N=10 data points per input-output pair
  • Multiple Testing: Benjamini-Hochberg FDR correction
  • Effect Size: Report correlation magnitude, not just significance
  • CI Display: 95% confidence intervals on all correlation values

# Toolbox Feature

**Purpose**: User's personal strategies, tactics, and best practices library

## Structure

- `pages/` - Main pages (ToolboxPage)
- `components/` - Feature-specific components (StrategyList, StrategyCard, StrategyForm)
- `hooks/` - Custom hooks (useStrategies)
- `services/` - Business logic and API calls (strategy.service.ts)
- `types.ts` - TypeScript types for this feature

## Responsibilities

- Personal strategy management
- Tactic organization
- Best practices capture
- Strategy search and categorization

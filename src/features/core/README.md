# Core Feature

**Purpose**: Shared infrastructure, authentication, and settings

## Structure

- `pages/` - Main pages (HomePage, SettingsPage)
- `components/` - Shared components (LoginScreen, ErrorBoundary, Navigation)
- `hooks/` - Shared hooks (useAuth, useDebounce, useLocalStorage)
- `services/` - Shared services (supabase.ts)
- `types.ts` - Shared TypeScript types

## Responsibilities

- User authentication
- Application settings
- Shared utilities
- Database connection
- Global error handling
- Common UI components

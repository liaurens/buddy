# Focus Feature

**Purpose**: Pomodoro timer and focus session tracking

## Structure

- `pages/` - Main pages (FocusPage)
- `components/` - Feature-specific components (PomodoroTimer, FocusSessionList, etc.)
- `hooks/` - Custom hooks (useFocusSession)
- `services/` - Business logic and API calls (focus.service.ts)
- `types.ts` - TypeScript types for this feature

## Responsibilities

- Pomodoro timer functionality
- Focus session tracking
- Break management
- Productivity analytics

# Tasks Feature

**Purpose**: Todo list management and quick note capture

## Structure

- `pages/` - Main pages (TodoPage, NotesPage)
- `components/` - Feature-specific components organized by subdomain (tasks, notes)
- `hooks/` - Custom hooks (useTasks, useNotes)
- `services/` - Business logic and API calls (task.service.ts, note.service.ts)
- `types.ts` - TypeScript types for this feature

## Responsibilities

- Task and todo list management
- Quick note capture with categorization
- Note organization and search
- Task completion tracking

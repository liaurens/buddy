# Planning Feature

**Purpose**: AI-powered daily planning, calendar integration, time tracking

## Structure

- `pages/` - Main pages (PlanPage, CalendarPage, ReflectionPage)
- `components/` - Feature-specific components (PlanGenerator, TimeBlockList, etc.)
- `hooks/` - Custom hooks (usePlan, useTimer, useReflection)
- `services/` - Business logic and API calls (planning.service.ts, ai.service.ts, calendar.service.ts, reflection.service.ts)
- `types.ts` - TypeScript types for this feature

## Responsibilities

- AI-powered daily plan generation
- Time block management and tracking
- Calendar integration
- Daily reflection and learning capture
- Activity templates
